Building Deepfake Detection Browser Extension

About Our Project

This browser extension will help users identify potential deepfakes by analyzing media and providing a confidence percentage regarding their authenticity. By leveraging machine learning techniques, we aim to assist users in making informed decisions about the content they encounter.

Technologies Used
